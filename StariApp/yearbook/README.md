# KSET yearbook 2019

## About

Ovo je ažurirana verzija KSET-ovog yearbooka nastala iz želje za generalizacijom prijašnje verzije te proširenjem iste
za dodavanje galerija budućih generacija.

Projekt je pisan u jezicima Python i HTML/CSS uz korištenje paketa Flask.

## Kako dodati novu generaciju?

Nova generacija se dodaje u yearbook kroz nekoliko koraka:

    -Obrađene fotografije staviti u folder /photos/<godina> (imati na umu da je omjer fotki 2:3)
    -U folder json/<godina> dodati json podatke za svaku sekciju (pratiti format prijašnjih godina)
    -U globalnoj listi YEARS u yearbook_app.py dodati string godine nove generacije
